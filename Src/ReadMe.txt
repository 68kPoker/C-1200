
This directory contains Source code written in C for Game engine.

 Sub-directories:
 - Init - initial code like system allocation, data loading,
 - Bobs - animation components,
 - Events - events (joystick, IDCMP, timer) handler,
 - Gadgets - graphisc user interface core,
 - Sfx - sound effects and music replaying,
 - Board - board engine for board games,
 - Edit - game editor.
